# Installing Snort3 on Ubuntu Server 24.04

1. Install Ubuntu Server
   1. I used VirtualBox and created a VM with 2GB RAM, 40GB HDD
   2. Set the NIC to be bridged during install so you can reach the internet to download packages and SSH to it when ready.
2. Update the packages
   1. `sudo bash`
   2. `apt update && apt upgrade -y`
3. Install SSH server
   1. `apt install openssh-server -y`
   2. `systemctl enable ssh`
   3. `systemctl start ssh`
4. SSH into server and install dependencies
   1. `sudo bash`
   2.  `apt install build-essential autoconf libtool autoconf-archive pkg-config cmake libdumbnet-dev flex bison libgoogle-perftools-dev libhwloc-dev libluajit-5.1-dev libssl-dev libpcap-dev libpcre2-dev zlib1g-dev -y`
5.  Install libDAQ (a dependency) from source
    1.  `cd /usr/local/src/`
    2.  `wget https://github.com/snort3/libdaq/archive/refs/tags/v3.0.23.tar.gz`
    3.  `tar -xvzf v3.0.23.tar.gz`
    4.  `cd libdaq-3.0.23`
    5.  `./bootstrap`
    6.  Configure
        1.  `./configure`
    7.  Compile
        1.  `make`
    8.  Install
        1.  `make install`
    9.  Update library cache
        1.  `ldconfig`
6.  Install Snort from source
    1.  `cd ..`
    2.  `wget https://github.com/snort3/snort3/archive/refs/tags/3.10.0.0.tar.gz`
    3.  `tar -xvzf 3.10.0.0.tar.gz`
    4.  `cd snort3-3.10.0.0`
    5.  `./configure_cmake.sh --prefix=/usr/local/snort`
    6.  `cd build`
    7.  Compile
        1.  `make -j$(nproc)`
        2.  The `-j$(nproc)` part tells make to use all your CPU cores for faster compilation.
    8.  Install
        1.  `make install`
7.  Verify installation
    1.  `/usr/local/snort/bin/snort -c /usr/local/snort/etc/snort/snort.lua -T`
